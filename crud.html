LES ÉTAPES

  • Installer Jsonserver : npm install -g json-server
  • Dans un terminal dans votre dossier : json-server --watch db.json
  • Modifier les données du fichier db.json
  • Lancer le server : localhost:3000/produits
  • Pour stoper le port 3000: npx kill-port 3000
